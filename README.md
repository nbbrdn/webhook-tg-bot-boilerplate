# Webhook Telegram Bot Boilerplate

## How to run

Without Docker:

```bash
BOT_TOKEN=token WEBHOOK_SECRET=my-secret BASE_WEBHOOK_URL=https://my-address WEB_SERVER_PORT=my-port python main.py
```
